# Requirements: THC v2.6 Extra Features Batch 10

**Defined:** 2026-01-28
**Core Value:** Risk must be required for progress. No tedious grinding to avoid challenge.

## v2.6 Requirements

### Soul Economy

- [ ] **SOUL-01**: Illagers drop soul dust at 20% rate (Pillager, Vindicator, Evoker, Illusioner, Ravager, Witch)
- [ ] **SOUL-02**: 4 soul dust crafts 1 soul soil (2x2 pattern)

### Spawn Distribution

- [ ] **SPAWN-01**: Deepslate region includes wither skeletons at 15% spawn weight
- [ ] **SPAWN-02**: Deepslate pillager weight reduced from 25% to 20%
- [ ] **SPAWN-03**: Deepslate vanilla fallback reduced from 45% to 35%

### Threat System

- [ ] **THRT-01**: Dealing X damage adds ceil(X/4) threat to mobs within 5 blocks of player
- [ ] **THRT-02**: Direct damage target does not receive double threat bonus

### World Interaction

- [ ] **WRLD-01**: Breaking gravel with shovel always drops flint
- [ ] **WRLD-02**: Mining fatigue exempts gravel
- [ ] **WRLD-03**: Mining fatigue exempts grass blocks (all types)
- [ ] **WRLD-04**: Mining fatigue exempts flower blocks (all types)
- [ ] **WRLD-05**: Mining fatigue exempts ore blocks (all types)
- [ ] **WRLD-06**: Mining fatigue exempts glass blocks
- [ ] **WRLD-07**: Mining fatigue exempts beds
- [ ] **WRLD-08**: Mining fatigue exempts all placeable-anywhere blocks (existing allowlist)

### Combat

- [ ] **CMBT-01**: Arrow hit speed effect reduced from Speed IV to Speed III
- [ ] **CMBT-02**: Melee pillager attack damage increased from 4.5 to 6.5

### Territory

- [ ] **TERR-01**: Villagers in claimed chunks do not count towards village registration
- [ ] **TERR-02**: Beds in claimed chunks do not count towards village registration

### Smithing

- [ ] **SMTH-01**: Smithing table upgrades leather armor to copper (using copper ingot count matching crafting recipe)
- [ ] **SMTH-02**: Smithing table upgrades copper armor to iron (using iron ingot count matching crafting recipe)
- [ ] **SMTH-03**: Smithing table upgrades iron armor to diamond (using diamond count matching crafting recipe)
- [ ] **SMTH-04**: Smithing table upgrades wooden tools to stone (using smooth stone count matching crafting recipe)
- [ ] **SMTH-05**: Smithing table upgrades stone tools to copper (using copper ingot count matching crafting recipe)
- [ ] **SMTH-06**: Smithing table upgrades copper tools to iron (using iron ingot count matching crafting recipe)
- [ ] **SMTH-07**: Smithing table upgrades iron tools to diamond (using diamond count matching crafting recipe)
- [ ] **SMTH-08**: Tier upgrades preserve all enchantments
- [ ] **SMTH-09**: Tier upgrades restore durability to maximum
- [ ] **SMTH-10**: Existing diamond â†’ netherite upgrade unchanged
- [ ] **SMTH-11**: Smithing table craftable with copper ingots (alternative to iron recipe)
