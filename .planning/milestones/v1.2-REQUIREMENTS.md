# Requirements: THC v1.2 Extra Features Batch 2

**Defined:** 2026-01-19
**Core Value:** Risk must be required for progress. No tedious grinding to avoid challenge.

## v1.2 Requirements

Requirements for v1.2 release. Each maps to roadmap phases.

### Parry Stun

- [x] **PARRY-01**: Successful buckler parry stuns enemies within 3 blocks (increased from current)
- [x] **PARRY-02**: Successful buckler parry applies ~1 block knockback to stunned enemies

### XP Economy

- [x] **XP-01**: XP orbs only spawn from mob deaths (and bottles o' enchanting)
- [x] **XP-02**: XP orbs blocked from mining ores
- [x] **XP-03**: XP orbs blocked from breeding animals
- [x] **XP-04**: XP orbs blocked from fishing
- [x] **XP-05**: XP orbs blocked from trading with villagers
- [x] **XP-06**: XP orbs blocked from smelting (furnace XP)

### Tiered Arrows

- [x] **ARROW-01**: Vanilla arrow renamed to "Flint Arrow" with flint_arrow.png texture
- [x] **ARROW-02**: Iron Arrow added (+1 damage, crafted via anvil: 64 flint + 1 iron ingot)
- [x] **ARROW-03**: Diamond Arrow added (+2 damage, crafted via anvil: 64 flint + 1 diamond)
- [x] **ARROW-04**: Netherite Arrow added (+3 damage, crafted via anvil: 64 flint + 1 netherite ingot)
- [x] **ARROW-05**: All tiered arrows use corresponding textures (iron_arrow.png, diamond_arrow.png, netherite_arrow.png)
