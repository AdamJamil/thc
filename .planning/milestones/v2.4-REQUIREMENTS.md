## v2.4 Requirements

Requirements for Extra Features Batch 8. All requirements complete.

### Boats & Mounts

- [~] **BOAT-01**: Iron boat item craftable (5 iron ingots + magma cream in minecart shape) — *Unverified*
- [~] **BOAT-02**: Iron boat uses custom texture (iron_boat.png) and icon (iron_boat_icon.png) — *Unverified*
- [~] **BOAT-03**: Iron boat works on lava the same way regular boats work on water — *Unverified*
- [~] **BOAT-04**: Iron boat item drop is lava-proof and floats (like netherite items) — *Unverified*
- [~] **BOAT-05**: Iron boat item flies towards player when boat is broken — *Unverified*
- [x] **SADL-01**: Saddles removed from all chest loot tables
- [x] **SADL-02**: Mobs no longer drop saddles
- [x] **SADL-03**: Saddles cannot be crafted (remove any existing recipe)

### Buckets & Fluids

- [x] **BUCK-01**: Copper bucket item craftable (standard bucket recipe with copper ingots)
- [x] **BUCK-02**: Copper bucket can only hold water or milk (not lava, powder snow, etc.)
- [x] **BUCK-03**: Copper bucket uses custom textures (copper_bucket.png, copper_bucket_of_water.png, copper_bucket_of_milk.png)
- [x] **BUCK-04**: Lava buckets cannot be placed (right-click does nothing)
- [x] **WATR-01**: Water placement from buckets creates non-source water at max height
- [x] **WATR-02**: Placed water has vanilla physics (flows and eventually disappears)

### Flight

- [x] **ELYT-01**: Fireworks no longer propel player during elytra flight
- [x] **ELYT-02**: Elytra gains 2x speed when diving
- [x] **ELYT-03**: Elytra loses 1.8x speed when ascending

### Brewing

- [x] **BREW-01**: Brewing stands removed from natural spawns (villages, igloos, etc.)
- [x] **BREW-02**: Brewing stands cannot be crafted
- [x] **BREW-03**: Potions removed from piglin bartering tables

### Combat/Armor

- [x] **ARMR-01**: Armor values rebalanced (leather 7, copper 10, iron 15, diamond 18, netherite 20)
- [x] **ARMR-02**: Toughness values rebalanced (leather 0, copper 0, iron 0, diamond 4, netherite 6)
- [x] **ARMR-03**: Per-piece distribution ensures each upgrade gives strictly more armor
- [x] **ARMR-04**: Half armor points allowed on individual pieces (bar rounds down)
