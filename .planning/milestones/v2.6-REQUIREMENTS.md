# Requirements: THC v2.6 Extra Features Batch 10

**Defined:** 2026-01-28
**Core Value:** Risk must be required for progress. No tedious grinding to avoid challenge.

## v2.6 Requirements

### Soul Economy

- [x] **SOUL-01**: Illagers drop soul dust at 20% rate (Pillager, Vindicator, Evoker, Illusioner, Ravager, Witch)
- [x] **SOUL-02**: 4 soul dust crafts 1 soul soil (2x2 pattern)

### Spawn Distribution

- [x] **SPAWN-01**: Deepslate region includes wither skeletons at 15% spawn weight
- [x] **SPAWN-02**: Deepslate pillager weight reduced from 25% to 20%
- [x] **SPAWN-03**: Deepslate vanilla fallback reduced from 45% to 35%

### Threat System

- [x] **THRT-01**: Dealing X damage adds ceil(X/4) threat to mobs within 5 blocks of player
- [x] **THRT-02**: Direct damage target does not receive double threat bonus

### World Interaction

- [x] **WRLD-01**: Breaking gravel with shovel always drops flint
- [x] **WRLD-02**: Mining fatigue exempts gravel
- [x] **WRLD-03**: Mining fatigue exempts grass blocks (all types)
- [x] **WRLD-04**: Mining fatigue exempts flower blocks (all types)
- [x] **WRLD-05**: Mining fatigue exempts ore blocks (all types)
- [x] **WRLD-06**: Mining fatigue exempts glass blocks
- [x] **WRLD-07**: Mining fatigue exempts beds
- [x] **WRLD-08**: Mining fatigue exempts all placeable-anywhere blocks (existing allowlist)

### Combat

- [x] **CMBT-01**: Arrow hit speed effect reduced from Speed IV to Speed III
- [x] **CMBT-02**: Melee pillager attack damage increased from 4.5 to 6.5

### Territory

- [x] **TERR-01**: Villagers in claimed chunks do not count towards village registration
- [x] **TERR-02**: Beds in claimed chunks do not count towards village registration

### Smithing

- [x] **SMTH-01**: Smithing table upgrades leather armor to copper (using copper ingot count matching crafting recipe)
- [x] **SMTH-02**: Smithing table upgrades copper armor to iron (using iron ingot count matching crafting recipe)
- [x] **SMTH-03**: Smithing table upgrades iron armor to diamond (using diamond count matching crafting recipe)
- [x] **SMTH-04**: Smithing table upgrades wooden tools to stone (using smooth stone count matching crafting recipe)
- [x] **SMTH-05**: Smithing table upgrades stone tools to copper (using copper ingot count matching crafting recipe)
- [x] **SMTH-06**: Smithing table upgrades copper tools to iron (using iron ingot count matching crafting recipe)
- [x] **SMTH-07**: Smithing table upgrades iron tools to diamond (using diamond count matching crafting recipe)
- [x] **SMTH-08**: Tier upgrades preserve all enchantments
- [x] **SMTH-09**: Tier upgrades restore durability to maximum
- [x] **SMTH-10**: Existing diamond â†’ netherite upgrade unchanged
- [x] **SMTH-11**: Smithing table craftable with copper ingots (alternative to iron recipe)
