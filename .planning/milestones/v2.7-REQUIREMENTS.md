# Requirements: THC v2.7

**Defined:** 2026-01-29
**Core Value:** Risk must be required for progress. No tedious grinding to avoid challenge.

## v2.7 Requirements

Requirements for Extra Features Batch 11. Each maps to roadmap phases.

### QoL Fixes

- [x] **QOL-01**: Mining fatigue does not increase beyond displayed level 10
- [x] **QOL-02**: Poise meter icons scaled ~7-10% smaller with spacing between them
- [x] **QOL-03**: Bells no longer have disabled ringing after dropping land plot
- [x] **QOL-04**: Experience bottles grant XP (restore from XP economy blocking)

### Combat Balancing

- [x] **CMBT-01**: Arrow crafting recipe yields 16 arrows (from 4)
- [x] **CMBT-02**: Pillager arrows deal 3-5 damage (from 5-7)
- [x] **CMBT-03**: Stray arrows deal 2-4 damage (from 4-8)

### Food Economy

- [x] **FOOD-01**: Bread recipe removed from game
- [x] **FOOD-02**: Dough item craftable (shapeless: 3 wheat + copper/iron water bucket, bucket preserved)
- [x] **FOOD-03**: Dough cooks into bread in furnace
- [x] **FOOD-04**: Pigs drop leather matching cow rates
- [x] **FOOD-05**: Sheep drop leather matching cow rates

### Enchantments

- [x] **ENCH-01**: Blast/fire/projectile/protection enchantments stackable on same armor
- [x] **ENCH-02**: Smite/sharpness/bane of arthropods stackable on same weapon

## Future Requirements

None — all identified features scoped to v2.7.
